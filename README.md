# Theme_Me-Mike_Boynton

Created by TheyCallMePapa and @Cariboudjan

Allows you to select a random Droptop theme and you can save up to eight custom themes and hot load them.
No need to install a custom theme again once it is saved.

Features

    Toggle Theme Previews
    Random theme every 24hrs , 1hr or 1 minute.
    Toggle random theme feature on/off
    Shows you the theme you are currently using.
